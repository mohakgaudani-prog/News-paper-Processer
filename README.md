# Daily Newspaper Processor

A personal web application for processing daily newspapers, extracting important business and investing news, and generating PDF reports with data visualization.

## Features

- **File Management**: Upload newspaper files (TXT, PDF, DOC, DOCX) via drag-and-drop or file browser
- **Article Processing**: Intelligent extraction and analysis of articles from newspaper files
- **AI-Powered Analysis**: Automatic categorization into business, investing, technology, health, and general categories
- **Importance Scoring**: AI-powered determination of article importance based on business/investing relevance
- **Data Visualization**: Interactive charts showing category distribution and daily article counts
- **Business Metrics Dashboard**: Real-time metrics including total articles, business-related content, and investment focus
- **Advanced Filtering**: Filter articles by category and search by keywords
- **PDF Export**: Generate professional PDF reports with filtered articles and metrics
- **Responsive Design**: Modern, mobile-friendly interface

## How to Use

1. **Upload Files**: Drag newspaper files into the upload zone or click "Browse Files"
2. **Process Articles**: Click "Process Articles" to analyze uploaded files
3. **Browse Results**: View categorized articles with business/investing scores
4. **Filter Content**: Use category filters and search to find specific articles
5. **Generate Reports**: Click "Export PDF" to create professional reports
